# 音楽制作ガイド — ポケモンの精神を継承する

このリポジトリで作成・修正する **すべてのBGMと効果音** は、ポケットモンスターの
BGM／効果音の精神を継承すること。新しい曲・SEを足すときも、既存曲を直すときも、
必ず以下の原則に沿わせる。（ユーザー恒久指示：2026-07 時点）

## 継承する「ポケモンの精神」

1. **PSG／チップチューンの音色を核にする**
   - リード・ベース・ハモリは矩形波（可変デューティ 12.5% / 25% / 50%）を基本にする。
     Web Audio では `PeriodicWave` でデューティ矩形波を作る。
   - パッド／やわらかい音は triangle（GBの波形メモリchの代わり）。
   - 打楽器はノイズ（バッファ乱数＋ハイパス/バンドパス）で作る。スネアは極短バースト。

2. **チャンネル分離の規律（GBの4ch精神）**
   - 「メロディ／ベース／ハモリ（アルペジオ）／打楽器」の役割を明確に分ける。
   - 音を盛りすぎない。各要素が聴き取れる隙間を残す。

3. **覚えやすい（口ずさめる）メロディ**
   - フック＋応答、モチーフの反復と変化。コードトーン中心＋弱拍で経過音。
   - 装飾よりも「歌える輪郭」を優先。

4. **快適な音域に置く**
   - リードはおおむね C5〜C6 を中心に。**G6 以上の刺さる高音を多用しない。**
   - 低音は濁らないよう整理する。

5. **ループ前提の構成**
   - 短いイントロ → 本体 → シームレスにループ。
   - A/B の対比や、2周目で要素を足すなどの起伏をつける。

6. **音を歪ませない・出しすぎない（クオリティの土台）**
   - マスターにブリックウォール・リミッター（`DynamicsCompressor` ratio≈20, threshold≈-2）を必ず入れる。
   - ディレイのフィードバックや残響は控えめに（feedback ≈ 0.2 以下）。時間経過で
     エコーが溜まって歪む事故を防ぐ。
   - マスター音量は控えめ（`vol/100 * 0.7` 程度）から始める。

7. **効果音もSEの精神で**
   - 攻撃・決定・キャンセル・カーソル・鳴き声などは、短く・ピッチ包絡（スイープ）と
     ノイズで作る。鳴き声は「1つの基本波形をシード違いでピッチ変化」させる方式。

## 実装メモ（このリポジトリの各ページ共通の作り）
- 音源ファイルは使わず、Web Audio API でリアルタイム合成する（1ファイル完結）。
- `voice()`（デチューン2音＋ローパス＋ADSR、`duty`でPSG矩形波、`fx`でディレイ/残響送り）を基本ブロックにする。
- 先読みスケジューラ（イントロを1回 → 本体をループ）で途切れなく再生する。
- 変更後は必ず、JS構文チェックと Web Audio/DOM モックの実行スモークテストを通す。

# 作曲の指針 — ポケモンDP × ジョン・ウィリアムズ（恒久ルール）

ユーザー恒久指示（2026-07）：**今後作成・修正するすべての曲に、以下の原理を必ず活かす。**
心に残るBGMの共通原理は「①覚えやすい短いメロディ最優先 ②長調を土台に“予想外の借用和音”で切なさを差す ③場面/人物ごとにテーマ音（ライトモチーフ）を決めて変奏で使い回す」。

## 1. メロディ第一主義
- **4〜5音の短い断片**を核にし、繰り返す＋少し変える。「ハミングできるか」で判定、できなければ複雑すぎ。
- 理論は後付け。まず口ずさめるフックを1つ決めてから和声・伴奏を足す。
- 跳躍（大きな音程の飛び）＝英雄性・ドラマ／なめらかな順次進行＝優しさ。

## 2. 「明るいのに切ない」を作る（DPの魔法）
- **長調を土台に、予想外の和音を1つだけ差す**とノスタルジーが出る：
  - **平行短調からの借用和音**（I → 借用の i → ♭VII。例: C → Cm → B♭ ＝ 216番道路）
  - **セカンダリードミナント**（V7/ii など。例: C → **A7** → Dm ＝ フタバタウン）
  - **付加音・拡張和音**（add9 / maj7 / 9th / ♯11）で余韻と煌めき
- 通常の終止（V→I）を**あえて避けて意外な和音へ跳ぶ**と“魔法/幻想”感（ヘドウィグのテーマ）。

## 3. 感情別コード進行レシピ（ハ長調基準・実音）
`I=C ii=Dm iii=Em IV=F V=G vi=Am`
- **冒険・出発**：C–G–Am–F（I–V–vi–IV）
- **郷愁・ほろ苦い（町）**：C–G–Am–F をゆったり＋ピアノ/鉄琴。深めるなら **C–A7–Dm–F**
- **切ない長調（雪道）**：C–Cm–B♭（I→借用i→♭VII）
- **不思議・幻想（湖/魔法）**：3度を避ける／Cadd9・Cmaj7・♯iv、意外な和音へ跳ぶ
- **緊張・戦闘/ボス**：短調で速く。Am–G–F–G（i–♭VII–♭VI–♭VII＝シロナ戦の型）
- **威圧・悪役**：短調＋反復音＋半音ずらした借用和音＋トライトーン移動＋低音強調（帝国のマーチ）
- **安心・回復**：短くシンプルな長調ループ。複雑にしない
- **勝利・凱旋**：長調で上向き旋律＋ホルン的音色。i–III–VII–VI（短調→長調へ抜ける）も有効

## 4. 楽器＝感情（音色そのものがメッセージ）
- ホルン/ブラス＝英雄性・気高さ／チェレスタ・鉄琴・ベル＝魔法・子供らしさ・郷愁
- ピアノ＋チャイム＋フルート＋弦＝“郷愁パレット”／高速スラップベース＝戦闘の緊張
- 低音金管・低いドローン＝脅威。制約（PSG/少ない音数）は「素朴で懐かしい」個性に変える。

## 5. ライトモチーフ（記憶の糊）
- **1曲1モチーフ**：主人公/町/大陸/ボスにテーマ旋律を決め、他シーンで**テンポ・調・楽器を変えて変奏**して再利用する。
- 物語の山場では、既存モチーフを「ゆっくり・短調・少ない楽器」で変奏＝記憶の呼び戻し。
- このリポジトリでは「ゴールド大陸」系（地上ハ長調／ダンジョンは相対イ短調）のように、**関連シーンを調性で連結**する。

## 6. ループの作法
- 曲の**始まりと終わりを同じ/調和する和音**に。ループ末尾に派手な盛り上げ・フィルを置かない（予測可能で疲れる）。
- 先読みスケジューラで「イントロ1回→本体ループ」。境目に残響を渡してシームレスに。

## 7. 判断基準（ベンチマーク）
- 30秒ループを**10回連続で聴いて飽きないか**（飽きる＝要素過多/メロが弱い）。
- **メロを口ずさめるか**。狙った感情と**キー/テンポ/楽器が一致**しているか。

## 8. 場面別チェックリスト
- 町/村＝長調・中庸・はっきりメロ・複雑にしない／ルート＝明るい行進＋歩くベース＋借用和音1つ
- 雑魚戦＝速く反復・短くループ（殺伐にしすぎない）／ボス＝短調・高速・駆け上がる旋律・印象的なピアノ導入
- ダンジョン＝静かで不穏・持続音・半音の揺れ／山場＝既存モチーフをゆっくり短調で変奏
