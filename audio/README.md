# 3章 ガーディアン大陸 BGM 音源

この `audio/` フォルダに、以下の **6つのmp3ファイル**を配置してください（半角の名前で）。
配置後、各プレイヤーページ／サウンドトラックのジュークボックスから、
**ブツ切れなしの無限シームレスループ**で再生されます。

| 置くファイル名 | シーン | 元の名前 |
|---|---|---|
| `guardian-landing.mp3`   | 上陸        | ガーディアン大陸　上陸 |
| `guardian-map.mp3`       | マップ      | ガーディアン大陸　マップ |
| `guardian-town.mp3`      | 街中        | ガーディアン大陸　街中 |
| `guardian-battle.mp3`    | 雑魚キャラ戦 | ガーディアン大陸　雑魚キャラ戦 |
| `guardian-midboss.mp3`   | 中ボス戦    | ガーディアン大陸　中ボス戦 |
| `guardian-greatboss.mp3` | 大ボス戦    | ガーディアン大陸　大ボス戦 |

## アップロード方法（GitHub Web画面）
1. GitHubでこのリポジトリを開く → `audio` フォルダに入る
2. **Add file → Upload files**
3. 6つのmp3を上の名前にリネームしてドラッグ&ドロップ
4. **Commit changes**（main ブランチ）
5. 1〜2分でGitHub Pagesに反映 → 各ページで再生可能

※ 完全にシームレスにするには、曲の**終わり→頭が自然につながるよう作られている**のが理想です。
　mp3のエンコード時に入る前後の無音は、プレイヤー側で自動トリムしています。
